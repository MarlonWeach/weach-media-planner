# Sample — Resposta de Mix de Mídia (Agente Media Planner)

Exemplo de resposta REAL gerada pela IA ao usar o prompt de mix.

Atenção:
- Percentuais sempre somam 100.
- Nenhum preço aparece.
- Apenas inventário válido.

============================================================
1. MIX GERADO (JSON)
============================================================

{
  "mix": [
    { "canal": "display_programatico", "percentual": 35 },
    { "canal": "social_programatico",  "percentual": 30 },
    { "canal": "video_programatico",   "percentual": 25 },
    { "canal": "cpl_cpi",              "percentual": 10 }
  ],
  "justificativa": "Para campanhas de leads no segmento automotivo, recomendamos uma combinação equilibrada entre volume de tráfego qualificado e estímulo visual ao usuário. O uso de Social impulsiona intenção imediata, enquanto Display e Vídeo ampliam alcance e sustentam frequência. Uma pequena fatia em CPL ajuda a reforçar entregas diretas."
}

============================================================
2. PONTOS DE REFERÊNCIA
============================================================

- Os 4 canais acima fazem parte do inventário Weach.
- A justificativa deve sempre explicar o porquê.
- A IA não pode incluir valores financeiros.
