# PRD - PBI-8: Precificação de Performance

## Contexto
A precificação de performance atualmente depende de avaliação manual caso a caso, considerando variáveis de mercado, histórico de campanha, qualidade de lead/conversão e experiência operacional.

## Problema
Não existe um fluxo estruturado no sistema para suportar essa análise manual especializada com rastreabilidade, resultando em risco de inconsistência e perda de contexto decisório.

## Objetivo
Criar base funcional para suportar precificação de performance assistida, mantendo flexibilidade para decisão humana e registrando critérios usados em cada cotação.

## Escopo Inicial
- Definir variáveis críticas para precificação manual de performance.
- Estruturar captura dessas variáveis no fluxo de cotação.
- Registrar racional e histórico de ajuste por cotação/formato.
- Disponibilizar base para futuras automações sem perder controle humano.

## Fora de Escopo (nesta fase)
- Automatização total da precificação de performance.
- Modelo preditivo de preço com aprendizado automático.

## Métricas de Sucesso
- 100% das cotações de performance com racional de preço registrado.
- Redução de divergência entre proposta e critério comercial esperado.
- Rastreabilidade de ajustes para auditoria/comparação posterior.
