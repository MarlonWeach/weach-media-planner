# Task 1-18: Implementar controle de preço e % budget no plano de mídia

## Status
✅ Done

## Descrição
Adicionar controles no plano de mídia para permitir ajuste operacional pelo usuário com regras de governança.

## Objetivo
Permitir otimização manual segura no passo final, respeitando restrições de negócio:
- preço só pode subir (nunca reduzir abaixo do sugerido),
- redistribuição de `% budget` por formato com atualização da visualização.

## Critérios de Aceite
- [x] Usuário pode editar preço por linha apenas para valores maiores que o preço sugerido original.
- [x] Tentativa de reduzir preço deve ser bloqueada com mensagem clara.
- [x] Usuário pode ajustar `% budget` por formato.
- [x] Soma de `% budget` deve permanecer 100% (com validação e feedback).
- [x] Ajustes devem atualizar totais e estimativas exibidas em tempo real.
- [x] Nova visualização deve refletir os ajustes antes de gerar PDF/salvar.
