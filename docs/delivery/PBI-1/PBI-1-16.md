# Task 1-16: Tornar distribuição de budget por formato guiada por IA

## Status
✅ Done

## Descrição
Evoluir o resultado do passo final para que a proporção de budget por formato não fique uniforme por padrão quando houver múltiplos formatos selecionados.

## Objetivo
Garantir distribuição de budget por formato coerente com o racional da IA e com as escolhas do usuário, mantendo restrições de canais/formato permitidos.

## Critérios de Aceite
- [x] Quando houver mais de um formato selecionado, a distribuição de `% budget` entre formatos não deve ser uniforme por padrão.
- [x] A distribuição deve respeitar os canais e formatos selecionados no wizard.
- [x] Se a IA retornar canais/formatos inválidos, aplicar fallback determinístico sem quebrar o plano.
- [x] Registrar claramente no código a estratégia de fallback para auditoria.
