# PBI-4: Fase 4 - Simulador Público

## Status
📌 Pendente

## Descrição
Criar página pública de simulador que permite que prospects façam simulações rápidas e se tornem leads para o time comercial.

## Objetivo
Gerar leads automaticamente através de uma ferramenta pública de simulação de planos de mídia.

## Entregáveis

### Página Pública
- ⏳ Página /simulador acessível sem login
- ⏳ Formulário simplificado
- ⏳ Design atrativo e profissional

### Lógica de Simulação
- ⏳ Estimativas genéricas (não sensíveis)
- ⏳ Range aproximado de custos
- ⏳ Mix básico de canais
- ⏳ Não expor preços internos

### Conversão em Lead
- ⏳ Captura de dados do usuário
- ⏳ Envio para CRM/Hub interno
- ⏳ E-mail automático com resumo
- ⏳ Notificação para time comercial

## Critérios de Aceite
- [ ] Acessível sem autenticação
- [ ] Não expõe dados sensíveis de pricing
- [ ] Gera leads qualificados
- [ ] Resposta rápida (< 3 segundos)
- [ ] Design responsivo

## Dependências
- PBI-1 (MVP Funcional - para lógica de cálculo)
